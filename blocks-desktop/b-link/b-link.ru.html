Ссылка

Блок со ссылкой. Используется в других блоках.

В простом случае для отображения блока необходимо задать следующий bemjson:

{ block: 'b-link', url: 'http:company.yandex.ru', title: 'Click here to learn more', target: '_blank', content: 'The best company all over the world' }

Свойство url преобразуется в атрибут href. Свойства title и target — в соотвествующие атрибуты.
